HTML stands for Hyper Text Markup Language.html is used to create web pages.a tag consists of left brace as well as right brace.html tags are of two types.they are paired tags and singular tags.a paired tag is a tag which has a closing tag at the end and singular tag is a tag which do not have a closing tag at the end.the some of the basic tags of html are html tag,head tag, title tag,body tag.

html tag represents root of an html document.and the head tag is used to define the head portion of the document which contains information related to the document.and the title tag defines the title of the document.and the body tag defines the main content of an html document.
